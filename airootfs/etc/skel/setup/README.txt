Запоминайте или фотографируйте!
Перед тем как что-либо делать, перенесите важные файлы на другой диск либо измените разделы с помощью редактора nano, открыв файл `install.sh` и заменив `sda1`, `sda2`, `sda3` на созданные. Хотя я рекомендую удалить все разделы и сделать всё по предложенному мной совету. Скрипт расчинан на свап, он должен быть создан после граба. Если не хотите чтобы был свап то введите sudo nano install.sh и уберите mkswap /dev/sda2 вместе с swapon /dev/sda2, после чего нажмите CTRL + O и CTRL + X. Если хотите поставить на SSD то измените в скрипте install.sh строку cfdisk /dev/sda на cfdisk /dev/ваш_ssd. Чтобы узнать свой SSD выйдите CTRL + X и введите команду lsblk и вы увидите nvme0n1, возмодно будет называться по другому, но не sda

Последовательность работы:

1. Вводим по очереди команды:

Для нормального отображения кириллицы:
```
setfont cyr-sun16
```

Для разрешения на запуск:
```
chmod +x install.sh
```

Запуск скрипта установки:
```
./install.sh
```

Разметка диска:
Удаляем все разделы, предворительно после того как перенесли важные файлы в другое место
Созданный файл с установочным скриптом был рассчитан на установку на HDD с тремя разделами: загрузчик, подкачка и система. Поэтому разметка следующая:

- Загрузчик (grub): 1 Gb (знаю, что можно и меньше, но так надёжнее).
- Подкачка (swap): выбирайте исходя из объёма оперативной памяти — можно от 4 Gb и больше, сколько не жалко.
- Корень системы (root): всё оставшееся пространство (если вы хотите пользоваться исключительно Arch'ем) либо 150 Gb (на пробу системы или для установки Windows рядом).

Выбираем `write` с помощью стрелок вправо (если на клавиатуре нет стрелок, то всё, габела) и нажимаем Enter.
Пишем ответ `yes` (без кавычек).

Где будут вопросы `y/n`, нажимаем `y` и Enter.

Перезагрузка:
После завершения будет перезагрузка. По умолчанию:
- пользователь — `user`,
- пароль — `1234` (для администратора или root тоже).

Вход в систему:
1. Входим с логином `user` и паролем `1234`.
2. После этого вводим команды:

Для нормального отображения кириллицы:
```
setfont cyr-sun16
```

Переход в каталог `/` и запуск установочного скрипта:
```
cd /
sudo chmod +x install-pkg.sh
```

(потребуется пароль, вводим `1234`)

Запускаем установочный скрипт:
```
./install-pkg.sh
```

Далее, по аналогии с предыдущей схемой, где `y/n`, жмём `y` и Enter до победного момента, пока не появится окно входа. После появления окна либо заходим с паролем по умолчанию, либо перезагружаемся, чтобы звук работал корректно (да, костыли).

Если после перезагрузки снова появилось окно входа в систему — поздравляю! Вы смогли установить Arch с минимальными усилиями. (Хоть есть и скрипт `archinstall`, но мне не понравилась разметка по секторам, поэтому я сделал свою).
